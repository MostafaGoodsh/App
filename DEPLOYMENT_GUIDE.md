# دليل نشر التطبيق مع Phantom Wallet

## 🚀 خطوات النشر

### 1. النشر على Lovable
```bash
# اضغط على زر "Publish" في أعلى يمين المحرر
# احصل على رابط التطبيق المنشور
```

### 2. ربط نطاق مخصص (اختياري)
- انتقل إلى Project > Settings > Domains
- أضف نطاقك المخصص
- اتبع تعليمات DNS

### 3. إعداد Supabase للإنتاج
```bash
# في لوحة تحكم Supabase:
# Authentication > URL Configuration
# أضف URL التطبيق المنشور كـ:
# - Site URL
# - Redirect URL
```

## 📱 تجربة المستخدم

### للمستخدمين الجدد:
1. **تثبيت Phantom Wallet**
   - زيارة phantom.app
   - تثبيت الإضافة للمتصفح
   - إنشاء محفظة جديدة

2. **استخدام التطبيق**
   - فتح التطبيق
   - الانتقال لصفحة المحفظة
   - النقر على "اتصال بـ Phantom"
   - الموافقة على الاتصال

### الميزات المتاحة:
- ✅ عرض الرصيد الحقيقي من Solana
- ✅ إرسال SOL بأمان
- ✅ عناوين محفظة حقيقية
- ✅ حماية المفاتيح الخاصة محلياً

## 🔒 الأمان

### نقاط القوة:
- المفاتيح الخاصة تبقى في محفظة المستخدم
- التطبيق لا يخزن معلومات حساسة
- المعاملات تتم عبر Phantom مباشرة

### للاستخدام الآمن:
- استخدم شبكة Devnet للاختبار أولاً
- ابدأ بمبالغ صغيرة
- تأكد من عناوين الاستقبال

# دليل إدارة الصفحة الرئيسية والبروفايل

## نظرة عامة
يوفر النظام المتقدم لإدارة الصفحة الرئيسية والبروفايل تحكمًا كاملاً في تخصيص المحتوى والتصميم.

---

## 🏠 إدارة بطاقات الصفحة الرئيسية

### الوصول
- **المسار**: `/admin/home-cards`
- **الصلاحيات**: مدير فقط

### الميزات الرئيسية

#### 1. إنشاء وتعديل البطاقات
- **أنواع البطاقات المتاحة**:
  - قياسي (Standard)
  - تعلم (Learning)
  - فيديوهات قصيرة (Reels)
  - تحديثات (Updates)
  - مهام (Tasks)
  - استدعاء (Callout)
  - هوية (Identity)
  - محفظة (Wallet)
  - أنوبيس (Anubis)
  - مخصص (Custom)

#### 2. تخصيص التصميم
```
- صور الخلفية: يمكن رفع صور مخصصة أو استخدام روابط
- الألوان: تحديد لون الخلفية ولون النص
- التدرجات: تطبيق تدرجات CSS مخصصة
- الأيقونات: إضافة أيقونات للبطاقات
```

#### 3. التحكم في الخطوط
- **نوع الخط**: Cairo, Tajawal, Amiri, Almarai, Arial
- **وزن الخط**: Normal, Bold, Light, Medium
- **أحجام منفصلة**:
  - حجم خط العنوان (صغير، متوسط، كبير، كبير جداً)
  - حجم خط المحتوى (صغير، متوسط، كبير)
  - حجم الخط العام

#### 4. المحتوى الداخلي
- **محتوى HTML**: إضافة صفحات كاملة بتنسيق HTML
- **محتوى متعدد اللغات**: دعم العربية والإنجليزية
- **المسارات**: تحديد روابط التوجيه لكل بطاقة

#### 5. الويدجتات الخارجية
أنواع الويدجتات المدعومة:
1. **Iframe مخصص**: أي موقع خارجي
2. **Dexscreener Chart**: رسوم بيانية للعملات
   ```
   مثال: https://dexscreener.com/solana/token-address
   ```
3. **Pump.fun Widget**: عرض معلومات العملة
   ```
   مثال: https://pump.fun/coin/token-address
   ```
4. **رصيد المحفظة**: عرض رصيد محفظة مباشر
5. **كود مخصص**: إضافة أي كود HTML/JavaScript

**إعدادات الويدجت (JSON)**:
```json
{
  "height": "400px",
  "width": "100%",
  "theme": "dark",
  "showHeader": true
}
```

#### 6. ترتيب البطاقات
- **أزرار الترتيب**: استخدام أسهم لأعلى/لأسفل لإعادة ترتيب البطاقات
- **الترتيب التلقائي**: يتم تحديث الترتيب فوراً
- **العرض حسب الترتيب**: البطاقات تظهر في الصفحة الرئيسية حسب `display_order`

#### 7. الحالة والظهور
- **تفعيل/إيقاف**: التحكم في ظهور البطاقة
- **معاينة مباشرة**: رؤية التغييرات فوراً

## 👤 إدارة إعدادات البروفايل

### الوصول
- **المسار**: `/admin/profile-settings`
- **الصلاحيات**: المستخدم نفسه (إعدادات شخصية)

### الميزات الرئيسية

#### 1. التخطيط (Layout)
- **أنواع التخطيط**:
  - قياسي (Standard): التخطيط الافتراضي
  - مضغوط (Compact): عرض مكثف للمعلومات
  - موسع (Extended): عرض تفصيلي كامل
  - بسيط (Minimal): عرض بسيط ونظيف

- **ترتيب الأقسام**: التحكم في ترتيب ظهور الأقسام
  - نظرة عامة (Overview)
  - الإحصائيات (Stats)
  - النشاط (Activity)

#### 2. التصميم (Styling)
```css
/* مثال على إعدادات التصميم */
صورة الخلفية: /lovable-uploads/profile-bg.jpg
لون الخلفية: #ffffff
التدرج: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
وضع السمة: light | dark | auto
```

#### 3. الخطوط (Typography)
- **نوع الخط**: اختيار من خطوط عربية مميزة
- **وزن الخط**: التحكم في سُمك النص
- **أحجام منفصلة**:
  - حجم خط العناوين
  - حجم خط المحتوى

#### 4. العرض (Visibility)
تحكم كامل في ما يظهر في البروفايل:
- ✅ عرض الإحصائيات
- ✅ عرض النشاط
- ✅ عرض إحصائيات المتابعة
- ✅ عرض قائمة المهام
- ✅ عرض روابط التواصل
- ✅ عرض تاريخ الانضمام

#### 5. الخصوصية (Privacy)
ثلاثة مستويات للخصوصية:
1. **عام (Public)** 👁️
   - مرئي للجميع
   - محركات البحث يمكنها الوصول
   
2. **للمتابعين فقط (Followers Only)** 👥
   - مرئي فقط للمتابعين
   - محمي من الوصول العام
   
3. **خاص (Private)** 🔒
   - مرئي فقط لك
   - أقصى درجات الخصوصية

## 📊 قاعدة البيانات

### جدول `home_page_cards`
```sql
الحقول الرئيسية:
- id: UUID
- title, title_en: العناوين
- description, description_en: الأوصاف
- slug: المعرّف الفريد
- card_type: نوع البطاقة
- display_order: ترتيب العرض
- is_active: حالة التفعيل
- background_image, background_color, background_gradient
- font_family, font_size, font_weight
- route_path: مسار التوجيه
- page_content, page_content_en: المحتوى الداخلي
- external_widget_url, widget_type, widget_config
```

### جدول `profile_customization`
```sql
الحقول الرئيسية:
- id: UUID
- user_id: معرف المستخدم
- layout_type: نوع التخطيط
- card_arrangement: ترتيب الأقسام (JSONB)
- background_image, background_color, background_gradient
- theme_mode: وضع السمة
- font_family, header_font_size, content_font_size
- show_stats, show_activity, show_follow_stats, show_todo_list
- external_widgets: (JSONB)
- profile_visibility: مستوى الخصوصية
```

## 🎨 أمثلة عملية

### مثال 1: إنشاء بطاقة مع ويدجت Dexscreener
```
1. انتقل إلى إدارة بطاقات الصفحة الرئيسية
2. اضغط "إضافة بطاقة جديدة"
3. في تبويب "أساسي":
   - العنوان: "عملة مصر"
   - Slug: "egypt-coin"
   - النوع: "custom"
4. في تبويب "الويدجتات":
   - نوع الويدجت: "Dexscreener Chart"
   - الرابط: https://dexscreener.com/solana/your-token
   - الإعدادات: {"height": "500px", "theme": "dark"}
5. احفظ التغييرات
```

### مثال 2: تخصيص بروفايل بسيط وأنيق
```
1. انتقل إلى إعدادات البروفايل
2. في "التخطيط":
   - اختر "Minimal"
3. في "التصميم":
   - لون الخلفية: #f8f9fa
   - وضع السمة: "light"
4. في "الخطوط":
   - نوع الخط: "Cairo"
   - حجم العناوين: "large"
5. في "العرض":
   - فعّل الإحصائيات والمتابعة فقط
   - أخفِ النشاط والمهام
6. في "الخصوصية":
   - اختر "public"
```

## 🔧 نصائح وأفضل الممارسات

### للصفحة الرئيسية:
1. **الترتيب المنطقي**: ضع البطاقات المهمة في الأعلى
2. **التناسق البصري**: استخدم نظام ألوان موحد
3. **الأداء**: تجنب الإفراط في استخدام الويدجتات الخارجية
4. **الويدجتات**: اختبر الويدجتات قبل نشرها للمستخدمين

### للبروفايل:
1. **الخصوصية أولاً**: ابدأ بإعدادات خصوصية صارمة
2. **التدرج في التخصيص**: جرّب التغييرات تدريجياً
3. **الخطوط**: استخدم خطوط واضحة وسهلة القراءة
4. **الصور**: استخدم صور عالية الجودة للخلفية

# دليل إدارة الصفحة الرئيسية والبروفايل المتقدم

## نظرة عامة
يوفر النظام الجديد إدارة شاملة ومتقدمة لبطاقات الصفحة الرئيسية وتخصيصات البروفايل مع دعم كامل للويدجتات الخارجية والتحكم في التصميم.

---

## 🏠 إدارة بطاقات الصفحة الرئيسية

### الوصول
- المسار: `/admin/home-cards`
- الصلاحيات: للمدراء فقط

### الميزات الرئيسية

#### 1. إدارة البطاقات
- **إنشاء بطاقات جديدة**: أضف بطاقات مخصصة بالكامل
- **تحرير البطاقات الموجودة**: عدل أي بطاقة في أي وقت
- **حذف البطاقات**: احذف البطاقات غير المطلوبة
- **إعادة ترتيب البطاقات**: استخدم أزرار السهم لتغيير ترتيب الظهور

#### 2. التخصيص الأساسي
- العنوان (عربي وإنجليزي)
- الوصف (عربي وإنجليزي)
- Slug (معرف فريد للبطاقة)
- نوع البطاقة (عادي، تعلم، ريلز، مهام، إلخ)
- مسار الصفحة (route)
- حالة النشاط (نشط/غير نشط)

#### 3. التصميم والستايل
- صورة الخلفية
- لون الخلفية
- Gradient الخلفية
- لون النص

#### 4. التحكم في الخطوط
- نوع الخط: Cairo, Amiri, Tajawal, Arial
- سمك الخط: عادي، عريض، خفيف
- أحجام الخطوط

#### 5. الويدجتات الخارجية
- iFrame
- DexScreener
- Pump.fun
- رصيد المحفظة
- كود مخصص

## 👤 إدارة تخصيصات البروفايل

### الوصول
- المسار: `/admin/profile-customization`
- الصلاحيات: للمدراء فقط

### الميزات
1. التخطيط: قياسي، مضغوط، موسع، بسيط
2. التصميم: خلفية، ألوان، سمة
3. الخطوط: النوع، الحجم، السمك
4. الخصوصية: عام، خاص، للمتابعين فقط
